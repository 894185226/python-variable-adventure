📖 什么是变量？

变量是程序中用来存储数据的容器。就像一个有标签的盒子：

  • 变量名：盒子上的标签，用来识别这个变量
  • 变量值：盒子里装的东西，即存储的数据
  • 数据类型：盒子里东西的类型（数字、文字等）
name = "小明"  # 字符串类型
age = 12      # 数字类型
score = 95.5  # 浮点数类型

✍️ 命名规则

给变量起名字需要遵守以下规则：

✅ 字母开头：my_name
❌ 数字开头：2name
✅ 下划线连接：user_name
❌ 空格：my name
✅ 区分大小写：Name ≠ name
❌ 关键字：class, if

📌 赋值符号 =

在Python中，= 不是数学中的"等于"，而是"赋值"的意思。

x = 5

把数字5放入名为x的盒子

x = x + 1

取出x的值(5)，加1后(6)再放回去
